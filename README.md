A complete betting tracker and profit calculator built with vanilla HTML, CSS, and JavaScript. Track your bets, build betslips with multiple matches, and calculate your path to profit using compound betting strategy.

## ✨ Features

### 📊 Dashboard
- View all your bets organized by status: **Online**, **Won**, and **Lost**
- Real-time statistics showing active bets, wins, losses, and total profit
- Switch between tabs to filter bets by status
- Update bet status (Won/Lost) directly from the dashboard

### 🧮 Profit Calculator
- Calculate how many consecutive winning days needed to reach your target profit
- Start with any stake amount and odds
- See final balance and progress visualization
- Perfect for the "1.5 odds daily to 1M TZS" strategy

### 🎫 Betslip Builder
- Add multiple matches to build your accumulator bet
- Select from common betting options:
  - Both Teams To Score (Yes/No)
  - Over/Under 1.5 & 2.5 Goals
  - 1X2 (Full Time & First Half)
- Enter custom odds for each match
- Automatic total odds calculation
- Real-time potential win display
- Place bets directly from the builder

### 💾 Data Persistence
- All data saved in browser's localStorage
- Bets remain after page refresh
- Export/backup ready (data stored as JSON)
